#数据结构
##stack
* 有push，pop，top，empty等方法
* STL中默认的实现是一个**双向队列**，可以使用基于向量或者表的实现，无论哪种实现，都会保证在常量时间内高效执行
* 添加**析构函数**的目的是防止数据结构在超出管理范围时存在垃圾没有回收，如下例：
```
for(int i = 0; i < 1000000; ++i) {
    Stack small;
	small.push(some_data);
}
```

##queue
* 有empty,push,pop,front,back,size等方法
* queue实现方法有循环数组，链表
* 保存在数组里的表称为顺序表
